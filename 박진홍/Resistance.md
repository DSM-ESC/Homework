# 저항

## 저항이란?
> 전기적 흐름을 강제로 방해하는 소자

* 저항의 종류
    1. DIP 타입 -> 다리 관통 후 땜질하여 붙히는 제품
    2. SIP 타입 -> 표면에 그대로 땜질하여 붙히는 제품

## 옴의 법칙
 > I = V / R  
   
![logo](https://t1.daumcdn.net/cfile/tistory/24107E37565657F825)

      I = 전류, V = 전압, R = 저항

>단위
>> A(암페어) : 전류에 대한 단위  
>> V(볼트)   : 전압에 대한 단위  
>> Ω(옴) : 저항에 대한 단위
