# LED (Light Emtting Diode)

## LED 구조

>1. PN 접합의 구조로 이루어짐

![i](http://study.zumst.com/upload/00-T33-00-43-06/%EC%BD%98%ED%85%90%EC%B8%A0%20%EC%88%98%EC%A0%95%20%EC%97%94%ED%8A%B8%EB%A6%AC%20345.jpg)
* P형 반도체 = 양(+)의 성분 포함  
* N형 반도체 = 음(-)의 성분 포함 
> 2. 화학물질  혹은 에너지 압력에 따라 색이 결정됨  
* 합성되는 재료  
    1. Ga (갈륨)
    2. As (비소)
    3. P (인)
    4. N (질소)

## 빛이 발생하는  이유

 >1. 전자(-)가 전공(+)으로 이동  
 * 정공 : 전자가 빠진 빈 공간  
 > 2. 에너지 발생
 * 전기 에너지 -> 열 에너지 -> 빛 에너지
